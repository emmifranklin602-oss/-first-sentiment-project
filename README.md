# First AI/ML Project: Sentiment Analysis Tool

Built in Google Colab using Hugging Face transformers library.

Analyzes text sentiment (POSITIVE or NEGATIVE) on topics related to community improvement and global goals.

Example texts and results:
- "Port Harcourt community needs better flood protection." → NEGATIVE (score \~1.00)
- "Nigeria's renewable energy projects are promising!" → POSITIVE (score \~1.00)
- "Improving education access is a global priority." → POSITIVE (score \~1.00)

Notebook file: sentiment-analysis-tool.ipynb

Goal: Practice AI to help analyze opinions for positive change.
